# D-JSON: Distributed JSON

## Quick start

```bash
npm install d-json
```

You may also clone the repo directly:
```bash
git clone https://github.com/Krinkle/D-JSON.git
```

Or [download the latest master](https://github.com/Krinkle/D-JSON/zipball/master).
